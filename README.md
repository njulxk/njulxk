# 日报周报收集汇总系统

## 功能特性

- 组员填写日报/周报
- 管理员查看、汇总、导出
- 自动定时汇总并发送Word文档

## 安装

```bash
npm install
```

## 配置

修改 `config.js` 文件，配置：
- 邮件服务器信息
- 管理员邮箱
- 端口号等

## 启动

```bash
npm start
```

开发模式：
```bash
npm run dev
```

## 访问

前端页面：http://localhost:3000
API接口：http://localhost:3000/api

## 默认账户

管理员：
- 用户名：admin
- 密码：admin123

## 定时任务

- 周二至周五上午9点：自动汇总前一日12点至今的日报
- 周一上午9点：自动汇总上周五12点至今的周报
